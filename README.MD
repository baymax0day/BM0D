# 内网漏洞扫描工具

## 目录介绍

    ├── Config.py       配置文件
    ├── lib             c语言链接库文件
    ├── network         网络发现模块[文件夹]
    ├── README.MD       本文档
    ├── run.py          自动化网络扫描
    ├── start.py        特定模块,具体网段扫描
    ├── temp            文件缓存目录[文件夹]
    └── vulnTool        漏洞模块
        ├── aspFrame    Web->asp[文件夹]
        ├── hardware    硬件,路由器等[文件夹]
        ├── jspFrame    Web->jsp[文件夹]
        ├── phpFrame    Web->php[文件夹]
        ├── server      服务器漏洞[文件夹]
        └── vulnScan.py 此目录的模块调用
